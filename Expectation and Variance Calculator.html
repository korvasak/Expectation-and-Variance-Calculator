Expectation and Variance Calculator

About this Calculator

This calculator helps you calculate the Expectation and Variance of a set of values and their probabilities.

Expectation (E[W]) is the average value, Expectation of Squared Values (E[W^2]) is the average of squared values, and Variance (Var[W]) is a measure of the spread of the values.

Use the input fields to enter values and their probabilities, then click "Calculate" to get the results.

You can add or remove input fields as needed using the "+" and "-" buttons.
